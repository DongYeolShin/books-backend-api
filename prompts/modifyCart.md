

# 1. 개발 목적
- 장바구니에 담긴 도서의 수량 변경 
- 장바구니에 담긴 도서의 삭제 


# 2. 기능 설명
## 2.1 도서의 수량 변경
- 선택된 도서의 수량 변경 
- end-point : /api/v1/carts
- method : patch
- 파라메터 양식 : JSON
- 파라메터 예시 :
```
  { "bookId" : "BOOK001",  "quantity" : 1 }
```
- 결과 데이터 
    - 성공 시 : { "code" : 200 , "message" : "수정되었습니다."}
    - 실패 시 : { "code" : 500 , "message" : "수정이 실패되었습니다."}

### 2.1.1 기능 설명
- 선택된 도서의 장바구니 수량 변경
- 변경될 수량으로 업데이트 
- 도서가 없을 경우 ERROR


## 2.2 도서의 삭제
- 선택된 도서를 장바구니에서 삭제 
- end-point : /api/v1/carts/{bookId}
- method : delete
- 결과 데이터
    - 성공 시 : { "code" : 200 , "message" : "삭제되었습니다."}
    - 실패 시 : { "code" : 500 , "message" : "삭제 실패되었습니다."}

### 2.2.1 기능 설명
- 선택된 도서를 장바구니에서 삭제
- 도서가 없을 경우 ERROR
